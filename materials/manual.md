# 設計ドキュメント

## クラス一覧

| クラス名 | 内容 |
|---------|------|
| Game | ゲーム全体の管理 |
| Player | プレイヤー操作 |
| Bomb | 爆弾 |
| Explosion | 爆風 |
| Enemy | 敵キャラクター |
| Item | 強化アイテム |
| Map | ステージ管理 |

---

# Game クラス

## 属性一覧

| 属性 | 内容 |
|------|------|
| player | プレイヤーオブジェクト |
| enemies | 敵のリスト |
| map | ステージ情報 |
| score | スコア |
| gameOver | ゲーム終了状態 |

## メソッド一覧

| メソッド | 内容 |
|----------|------|
| startGame() | ゲーム開始 |
| update() | 毎フレームの処理 |
| checkGameOver() | ゲーム終了判定 |

## メモ

- ゲーム全体を管理するクラス
- プレイヤーや敵、爆弾の動きをまとめて処理する

---

# Player クラス

## 属性一覧

| 属性 | 内容 |
|------|------|
| x | 横方向の位置 |
| y | 縦方向の位置 |
| speed | 移動速度 |
| bombCount | 設置できる爆弾数 |
| alive | 生存状態 |

## メソッド一覧

| メソッド | 内容 |
|----------|------|
| move() | プレイヤー移動 |
| placeBomb() | 爆弾設置 |
| takeDamage() | 爆風を受ける |
| die() | 死亡処理 |

## メモ

- プレイヤーキャラクターを表す
- キーボード入力で移動する

---

# Bomb クラス

## 属性一覧

| 属性 | 内容 |
|------|------|
| x | 爆弾の位置 X |
| y | 爆弾の位置 Y |
| timer | 爆発までの時間 |
| power | 爆風範囲 |
| exploded | 爆発済みか |

## メソッド一覧

| メソッド | 内容 |
|----------|------|
| countdown() | タイマー減少 |
| explode() | 爆発処理 |

## メモ

- プレイヤーが設置する爆弾
- 一定時間後に Explosion を生成する

# Explosion クラス

## 属性一覧

| 属性 | 内容 |
|------|------|

| y | 爆発中心位置 |
| range | 爆風の長さ |

## メソッド一覧

| メソッド | 内容 |
|----------|------|

- 範囲内のキャラクターにダメージ

# Map クラス

## 属性一覧

| 属性 | 内容 |
|------|------|
| width | マップ横幅 |
| height | マップ縦幅 |

## メソッド一覧

| メソッド | 内容 |
|----------|------|
| createMap() | マップ生成 |
| checkCollision() | 壁との衝突判定 |

## メモ

- ステージの管理
- 爆風や移動できる場所を判断する
